ПОЧЕМУ SSL HANDSHAKE FAILS - ТЕХНИЧЕСКОЕ ОБЪЯСНЕНИЕ:
================================================================================

Windows curl использует Schannel (Windows native SSL library).

Schannel НЕ ПОДДЕРЖИВАЕТ:
- ГОСТ Р 34.10-2012 (цифровая подпись)
- ГОСТ Р 34.11-2012 (хеш Стрибог)  
- GOST cipher suites

GOST API ТРЕБУЕТ:
- Сертификат от УЦ КриптоПРО
- OpenSSL с GOST engine
- curl скомпилированный с GOST OpenSSL

Пытаться открыть китайский сайт с SM2/SM3 шифрованием
в обычном браузере - не получится без драйвера.
