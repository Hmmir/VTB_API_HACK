⚡ БЫСТРЫЙ СТАРТ
================================================================================

1. Установить зависимости:
   pip install -r requirements_analysis.txt

2. Запустить backend:
   docker compose up -d

3. Запустить анализ:
   python analyze_all_users.py

4. Открыть результаты:
   explorer analysis_output
