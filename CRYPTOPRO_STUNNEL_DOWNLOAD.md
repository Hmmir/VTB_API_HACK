- **Контрольная сумма ГОСТ**: `1D8047302EB7E56FEB2459790745113AC81E6ACEDE962EC6B2E172E22B793AF3`

## Как это работает:

```
Ваше приложение → stunnel (localhost:8444) → GOST TLS → api.gost.bankingapi.ru:8443
                     ↓
              КриптоПро CSP (ГОСТ алгоритмы + сертификат)
```

### Настройка stunnel.conf:

```ini
[gost-api]
client = yes                                    # МЫ - КЛИЕНТ
accept = 127.0.0.1:8444                        # Локальный порт
connect = api.gost.bankingapi.ru:8443          # GOST API
engineId = capi                                 # Используем КриптоПро через CAPI
```

### После установки stunnel:

1. **Запустить stunnel** с конфигурацией
2. **В вашем коде** подключаться к `http://127.0.0.1:8444` вместо `https://api.gost.bankingapi.ru:8443`
3. **stunnel автоматически** установит GOST TLS соединение через КриптоПро CSP
